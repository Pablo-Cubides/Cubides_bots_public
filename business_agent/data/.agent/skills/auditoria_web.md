# SKILL: Auditoría Web
**Descripción**: Habilidad para revisar el estado, SEO técnico y tiempo de respuesta de las páginas web de Primary User.

## Comandos permitidos (CLI/Internos)
- `curl_check`
- `seo_analyzer`
- `lighthouse_cli`

## Ejecución
1. Leer lista de dominios desde la base de conocimientos (`findings.md`).
2. Realizar ping/curl a cada dominio para verificar uptime.
3. Registrar métricas de rendimiento.
4. Si hay caídas, registrar crítica en `errors.md` y alertar en `progress.md`.
5. Requerimiento: Aprobación de Primary User NO requerida (Solo lectura).
