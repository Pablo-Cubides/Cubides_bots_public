# SKILL: Análisis de Tendencias
**Descripción**: Extrae información de fuentes RSS, APIs de noticias y tendencias de búsqueda para identificar oportunidades de negocio.

## Comandos permitidos (CLI/Internos)
- `fetch_news`
- `scrape_trends`

## Ejecución
1. Consultar tendencias diarias.
2. Filtrar por nichos de interés de Primary User.
3. Formatear los 3 hallazgos más importantes.
4. Escribir resultados en `findings.md` bajo la cabecera `## Tendencias de Mercado`.
5. Requerimiento: Aprobación de Primary User NO requerida (Solo consumo de API gratuita).
