# Personal Agent Memory Template

This public template intentionally contains no real personal memory.

Use this file only for generic, non-sensitive defaults. In a real deployment, keep private memory in ignored files, encrypted files, or a private repository.

## Stable Technical Notes

- The personal agent runs through Claude Code CLI.
- Slack is the normal conversational channel.
- Notion can store structured records such as habits, workouts, meals, expenses, and social interactions when configured.
- Google Workspace can store long-form reports after OAuth verification.
