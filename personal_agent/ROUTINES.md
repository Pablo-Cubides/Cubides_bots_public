# Personal Agent Routines Template

## Morning Routine

Prepare a conversational plan for the day using pending tasks, health/habit priorities, productivity goals, and approved improvements.

## Night Routine

Ask how the day went, summarize meaningful progress, identify blockers, and preserve only useful non-sensitive memory in the configured private storage.

## Weekly Routine

Prepare a review of patterns, wins, blockers, and proposed improvements. Keep raw personal details out of git.
