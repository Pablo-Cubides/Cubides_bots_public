## Regla Editorial

- `README.md` en raíz es la portada ejecutiva.
- `docs/current-state.md` describe el estado vivo.
- `docs/operations-runbook.md` tiene comandos.
- Las guías específicas explican configuración y contexto.
- No documentar secretos reales, solo nombres de variables y presencia esperada.
